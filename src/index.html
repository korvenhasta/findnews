Об авторе

Меня зовут Марина Кенина. В прошлом — support specialist в компании Mindbox. С мая по декабрь 2019 года — студент курса по фронтенд-разработке Яндекс.Практикума. Это мой дипломный проект. В нем использованы технологии: HTML, CSS, JavaScript, Git, Webpack, а также подход BEM.

В течение курса Яндекс.Практикум сверстала простой одностраничный сайт, сделала адаптивную верстку, реализовала два проекта на ванильном JavaScript со сборками через Webpack. Проекты обмениваются данными с API. Работы проходили ревью, в том числе через pull request на Github.
